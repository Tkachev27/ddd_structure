## Description
Создает тултип с заданным текстом. Представляет из себя замену материаловской директивы и копирует ее поведение.
В зависимости от свободного места тултип меняет координаты и может отображаться вверх, либо вниз.

## Usage
Используется у тегов, в директиву передается текст тултипа.

<Source
  language='html'
  dark
  format={true}
  code={`
    <span [tooltip]="getNodeTooltip(child) | async">
  `}
/>

## Необходимо добавить
Для поддержки полного функционала материаловской диррективы необходимо добавить слеудющее:

1. Поддержку разных позиций тултипа: 'left' | 'right' | 'above' | 'below' | 'before' | 'after' (пока есть только below/above).
2. Поддержку классов (сейчас как дефолтный выбран класс pre-line).
3. Возможность задать delay перед появление тултипа.
4. Небольшую анимацию перед закрытием тултипа.
5. Возможность задать disabled.
