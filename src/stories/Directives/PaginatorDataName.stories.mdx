<Meta title="Directives/PaginatorDataName" />

## Description
Директива для добавления аттрибутов **data-name** в пагинатор таблицы. 

|Элемент|Селектор      |data-name|
|-------|--------------|---------|
|Селектор выбора количества элементов на странице|`<mat-select></mat-select>`|data-name="paginator-selector"|
|Информация о количестве элементов|`<div class="mat-paginator-range-label"></div>`|data-name="paginator-range"|
|Кнопка перемещения в начало списка|`<button></button>`|data-name="paginator-first-page"|
|Кнопка перехода к следующей странице|`<button></button>`|data-name="paginator-next-page"|
|Кнопка перехода к предыдущей странице|`<button></button>`|data-name="paginator-previous-page"|
|Кнопка перемещения в конец списка|`<button></button>`|data-name="paginator-last-page"|
|5|`<mat-option></mat-option>`|data-name="paginator-option-5"|
|10|`<mat-option></mat-option>`|data-name="paginator-option-10"|
|25|`<mat-option></mat-option>`|data-name="paginator-option-25"|
|100|`<mat-option></mat-option>`|data-name="paginator-option-100"|

## Usage
Импортировать в ваш модуль, модуль директив:
<Source
  language='js'
  dark
  format={true}
  code={dedent`
    import { DirectivesModule } from 'src/app/shared/directives/directives.module';
  `}
/>

Использовать в пагинаторе таблицы:
<Source
  language='html'
  dark
  format={true}
  code={`
    <mat-paginator
        appPaginatorDataName>
    </mat-paginator>
  `}
/>

Так же можно добавить префикс для значения data-name (на выходе получим - `table-group-paginator-selector`):
<Source
  language='html'
  dark
  format={true}
  code={`
    <mat-paginator
        appPaginatorDataName="table-group">
    </mat-paginator>
  `}
/>
