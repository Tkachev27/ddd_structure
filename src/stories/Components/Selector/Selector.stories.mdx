Селектор выбора интервала обновления

## Usage
Импортировать в модуль компонента:
<Source
  language='ts'
  dark
  format={true}
  code={dedent`
    import { IntervalSelectorModule } from 'src/app/shared/components/interval-selector/interval-selector.module';
  `}
/>

Использовать в компоненте:
<Source
  language='html'
  dark
  format={true}
  code={dedent`

        [updateIntervals]="UPDATE_INTERVALS"
        [defaultInterval]="DEFAULT_INTERVAL_INDEX"

  `}
/>
