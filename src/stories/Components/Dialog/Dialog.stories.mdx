## Usage
Ипортировать сервис ошибок
<Source
  language='typescript'
  dark
  format={true}
  code={dedent`
    import { ErrorsFromServerService } from 'src/app/shared/services/errors-from-server.service';
  `}
/>

Объявить в конструкторе
<Source
  language='typescript'
  dark
  format={true}
  code={dedent`
    constructor(
      private errorFromServerService: ErrorsFromServerService) {
    }
  `}
/>

Использовать в компоненте
<Source
  language='typescript'
  dark
  format={true}
  code={dedent`
    error => {
      this.errorService.showErrorsInDialog(error.error, InformationalMessages.getInterfaces);
    }
  `}
/>
